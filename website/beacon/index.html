Get to know Beacon.

Recents that actually work.

Spotlight and Finder Recents miss freshly saved files all the time. Beacon scans your filesystem directly — so screenshots, downloads, and exports appear the moment they land on disk.

Fast lane for fresh files.

Version 1.0.3 adds a priority scan of Downloads and Desktop before any deep crawl, so your latest screenshot is always at the top.

Search by name and content.

Beacon uses Spotlight where it's strong — file names and document content inside PDFs, Office files, and iWork docs. Content matches kick in at 3+ characters and are flagged with a "text match" badge.

Smart ranking.

Exact name matches and your own files rank first. Folders surface reliably. Every source is case- and accent-insensitive — "jose" finds "José."

Your data.
Just on your Mac.

Beacon is a local search tool. File search works out of the box with no permissions. Messages, Notes, and Safari history read databases on your machine — nothing is sent to a server.

You control access.

Full Disk Access is optional and only unlocks Messages, Notes, Safari history, and protected folders. Grant it once in System Settings when you're ready.

Setup guide

Fully keyboard-driven.

Summon Beacon with Option + S from any app. Navigate results with arrow keys, open with Return, preview with ⌘Y, reveal in Finder with ⌘Return, and copy paths with ⌘C.

Tab through filters.

Press Tab or Shift+Tab to cycle type filters — All, Recents, Apps, Messages, Notes, History, and more — without touching the mouse.

Quick Look

Native Swift.
Zero dependencies.

Beacon is built with Swift and SwiftUI — a lightweight menu-bar app with a floating search overlay. No Electron, no cloud accounts, no subscription.

Signed and notarized.

Every release is signed with a Developer ID and notarized by Apple. Double-click the DMG, and Beacon installs itself into Applications.

Why Beacon?

Spotlight got cluttered. Finder got confusing.
Beacon stays small and local.

Spotlight became a kitchen sink.

Local files mixed with web suggestions, Siri cards, and dictionary results. Beacon keeps search focused — files, apps, and your data, ranked and stable as you type.

No index lag.

When Spotlight's index falls behind, files on your Desktop vanish. Beacon's Recents scanner reads the filesystem directly.

Finder Recents lie.

A download you haven't opened yet? Missing from Finder Recents. Beacon sees it the moment it hits disk.

One overlay.
Every source.

Messages, Notes, Clipboard, and History each have focused filters — so they never drown out file search.

All Recents Apps Messages

Built for macOS 13+.

Native SwiftUI, global hotkey via Carbon API (no Accessibility permission needed), Quick Look previews, and Developer ID signed releases. Zero third-party dependencies.

How it works

Download Beacon for Mac.

Version 1.0.3 · macOS 13 or later · Free

Download Release notes
  1. Download Beacon-1.0.3.dmg and open it.
  2. Double-click Beacon — it installs into Applications and opens with a hotkey tip.
  3. Press Option + S anywhere to search. Beacon lives in your menu bar.

Open on GitHub.

Beacon is open source. Clone the repo, run bash scripts/run.sh, and you're searching in under a minute.

View repository Documentation Report an issue

Beacon requires macOS 13 or newer. File search works immediately. Messages, Notes, and Safari history require Full Disk Access — a one-time toggle in System Settings → Privacy & Security.
